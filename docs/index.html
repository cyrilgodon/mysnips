Privacy Policy — Last updated April 2026

✓ No data collected

What data does MySnips collect?

None. MySnips does not collect, store, transmit, or share any personal data or usage data.

Where are my snippets stored?

Your snippets are stored exclusively in your browser using chrome.storage.local. This data lives only on your device and is never sent anywhere.

Does MySnips connect to any server?

No. MySnips makes zero network requests. There is no backend, no analytics, no telemetry, no third-party service of any kind.

Why does MySnips request access to all websites?

The permissions <all_urls> and scripting are required solely to inject the snippet panel into the page and insert text directly into the focused field — on whichever website you are visiting. These permissions are never used to read page content, track your browsing, or transmit any information.

Is the code open source?

Yes. The full source code is available on GitHub. Anyone can read and verify it.
